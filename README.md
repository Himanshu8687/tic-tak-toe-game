Tic Tac Toe Game 🎮

This is a simple Tic Tac Toe web game built using HTML, CSS, and JavaScript.
The game allows two players to play turn by turn on a 3×3 grid.

📌 Features

- 3×3 Tic Tac Toe board
- Two-player gameplay
- Win detection logic
- Reset game button
- Simple and clean UI

🛠️ Technologies Used

- HTML
- CSS
- JavaScript

🚀 How to Play

1. Player X starts the game.
2. Players take turns clicking on the boxes.
3. The first player to align 3 marks in a row, column, or diagonal wins.
4. Click Reset Game to start a new match.
